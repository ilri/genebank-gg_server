GRIN-Global Full Installer CD

This CD contains the following software:

  1. Prerequisite software required by GRIN-Global to operate properly.
  2. All GRIN-Global Server and Client Installers
  3. Example GRIN-Global Data and Index Files

For more information, please view the Read Me Documentation. Note: Requires Adobe Reader.

  1. Install Prerequisites

    Before any GRIN-Global applications will function properly, you must install the prerequisite software.

    Server Prerequisites

    If you will be installing the GRIN-Global Server applications on this machine, install the Server Prerequisites.

    Client Prerequisites

    If you will be installing the GRIN-Global Client applications on this machine, install the Client Prerequisites.

  2. Install GRIN-Global Updater

    GRIN-Global includes an Updater application. This is used to install all other GRIN-Global applications. It can also be used to get the latest releases from a remote server. Please install the GRIN-Global Updater before any other GRIN-Global applications.

  3. Run GRIN-Global Updater

    Locate the GRIN-Global Updater launch icon in your Windows Start menu. It should be at GRIN-Global / GRIN-Global Updater. Double click it to launch the GRIN-Global Updater.

    If Installing the Server Applications...

    1. Make sure your 'Download From Server:' is pointing at the appropriate resource.
    2. Click 'Check For Server Updates'
    3. The grid will fill with four applications. Install all four as needed by clicking 'Download / Install'

    If Installing the Client Applications...

    1. Make sure your 'Download From Server:' is pointing at the appropriate resource.
    2. click 'Check For Client Updates'
    3. The grid will fill with one application. Install it by clicking 'Download / Install'.
